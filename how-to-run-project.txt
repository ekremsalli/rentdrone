Version:Node 18,Python 3,Django 3

First run django then react app
Run command step by step for django
1-"pip install -r requirements.txt"
2-"python manage.py makemigrations"
3-"python manage.py migrate"
4-"python manage.py runserver"
Run command step by step for react
1-"npm install"
2-"npm start"


Dockerize
docker-compose up -d
